## Features
- Student authentication
- Submit and manage applications
- Admin dashboard with approve/reject functionality
- File upload (CV, Academic Record, Matric Certificate, ID Document)

## Technology Stack
- Flutter
- Supabase (Backend)
- Provider (State Management)

## Setup Instructions
1. Clone the repository
2. Run `flutter pub get`
3. Configure Supabase credentials
4. Run `flutter run`
